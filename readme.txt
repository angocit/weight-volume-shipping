== Description ==

Weight and volume Based Shipping is a simple yet flexible shipping method for WooCommerce focused mainly on the order weight and volume to calculate shipping cost. 

== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Weight Volume Shipping'
3. Activate 'Weight Volume Shipping' from your Plugins page.

=== Manually ===

1. Upload the `weight-volume-shipping` folder to the `/wp-content/plugins/` directory
2. Activate the 'Weight Volume Shipping' plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.

==== SETUP=====
1. Go to WooCommerce->Setting->Shipping.
2. Choose Zone name and click edit.
3. In Shipping methods Click Add shipping method then choose Weight Volume Shipping.
4. Click tab Weight Volume Shipping to setting your shipping method.
